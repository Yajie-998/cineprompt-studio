# CinePrompt Studio

AI电影分镜提示词工作台 - Windows桌面应用程序

## 项目简介

CinePrompt Studio是一款专为电影制作和AI艺术创作设计的桌面应用程序，帮助用户高效生成和管理AI绘画提示词。

## 核心功能

### 1. CineStyle Engine（风格引擎）
- 风格预设库：内置多种电影风格预设（电影质感、赛博朋克、复古怀旧等）
- 维度调音台：四个风格维度可自由调节
  - 光影风格：自然光、影棚光、电影感光效、黄金时刻、霓虹光
  - 镜头语言：广角镜头、长焦镜头、微距镜头、荷兰角、跟拍镜头
  - 胶片质感：柯达胶片、富士胶片、宝丽来、黑白胶片、颗粒感
  - 渲染风格：照片级写实、插画风格、3D渲染、水彩风格、动漫风格
- 保存/加载：支持保存自定义预设，自动加载历史记录

### 2. 分镜生产车间
- Excel式在线表格编辑器
- 支持字段：序号、景别、角度、画面描述、提示词（中英文）
- 批量导入：支持CSV格式导入
- 实时预览：右侧面板实时显示分镜内容
- 锁定机制：支持锁定单行提示词，防止误修改
- 导出功能：支持导出为CSV格式

### 3. 智能拆解模块
- 文本输入：支持输入剧本或场景描述
- AI拆解：使用大语言模型自动拆解为分镜
- 支持模型：
  - 智谱GLM（默认）
  - 通义千问
  - 豆包
  - 自定义端点
- 结果导入：支持单个导入或全部导入到分镜生产车间
- 结果导出：支持导出为JSON格式

### 4. 提示词管理台
- 从分镜加载：一键从分镜生产车间加载所有提示词
- 批量操作：
  - 批量锁定/解锁
  - 批量翻译（中英互译）
  - 批量删除
- 多格式导出：
  - CSV
  - TXT
  - Markdown
  - JSON

## 技术栈

- **框架**: Electron + Vue3 + TypeScript
- **构建工具**: Vite
- **UI**: 自定义暗色主题
- **数据存储**: localStorage

## 使用说明

### 首次使用

1. 启动应用后，左侧导航栏可选择不同功能模块
2. 推荐使用流程：
   - 在"分镜生产车间"录入分镜信息
   - 在"CineStyle Engine"选择风格并复制提示词
   - 在"智能拆解模块"使用AI自动拆解剧本
   - 在"提示词管理台"批量管理和导出提示词

### API配置

1. 进入"智能拆解模块"
2. 点击"设置"按钮
3. 选择模型提供商（智谱GLM、通义千问、豆包或自定义）
4. 输入API密钥
5. 保存设置

### 数据存储

- 所有数据自动保存到浏览器localStorage
- 分镜数据在"分镜生产车间"和"提示词管理台"之间共享
- 风格预设自动保存和加载

## 注意事项

1. **API密钥安全**：API密钥保存在本地localStorage，请勿泄露
2. **网络要求**：智能拆解功能需要网络连接调用大语言模型API
3. **数据备份**：建议定期导出重要数据作为备份
4. **CSV导入**：确保CSV文件使用UTF-8编码，字段顺序为：序号、景别、角度、画面描述、提示词（中文）、提示词（英文）

## 系统要求

- Windows 10或更高版本
- Node.js 18或更高版本（开发环境）
